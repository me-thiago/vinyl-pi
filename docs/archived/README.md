## Política de Arquivamento

Documentos são movidos para cá quando:

1. **Superseded**: Uma versão mais recente existe
2. **Resolvido**: Problema documentado foi corrigido
3. **Desatualizado**: Informações não refletem mais o estado atual
4. **Consolidado**: Conteúdo foi integrado em outro documento

Documentos **nunca são deletados** - sempre arquivados para manter histórico de decisões.
